xmas-generator:

This is a simple application created during the December 2022 coding week at Boolean accademy. The app allows users to change the name displayed on the screen, as well as the background image and message, by clicking on the image or text.

Getting Started:
To use the app, simply open the index.html file in a web browser. The app will load and display a default message with a background image.
